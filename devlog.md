### 🐛 已知问题
- [ ] **明星真人头像加载不稳定** — 尝试过以下方案均未完全解决：
  - Wikipedia REST API（浏览器直连）：部分显示，多位明星时仅首张加载
  - Wikipedia MediaWiki API（服务端代理）：同上
  - 百度图片搜索（服务端）：JSON 接口解析失败
  - Wikidata（服务端代理）：Vercel 服务器访问受限
  - **当前方案**：CSS 渐变头像 + 浏览器直连 Wikipedia（600ms 间隔），失败时保留 CSS 头像
  - **根本原因**：Wikimedia 系列 API 对 Vercel IP 段/高频请求有限流
  - **可能的解决方向**：预置明星照片数据库、Bing Image Search API、或使用国内可用的图片 CDN
