## 📷 阶段 3：拍照与上传功能

### 目标
实现拍照和相册选择，图片预览和 Base64 转换。

### 任务清单

| # | 任务 | 说明 |
|---|------|------|
| 3.1 | 相机拍照功能 | `<input type="file" capture="environment">` |
| 3.2 | 相册选择功能 | `<input type="file" accept="image/*">` |
| 3.3 | 图片预览 | FileReader 读取并显示 |
| 3.4 | 图片压缩 | Canvas 压缩至长边 ≤ 2048px |
| 3.5 | Base64 编码 | 转为 Data URL 格式 |
| 3.6 | 重新选择 | 清除预览，重新拍照/选图 |
| 3.7 | 开始分析按钮 | 照片就绪后点亮按钮 |

### 完成标准
- [ ] 可调用相机拍照
- [ ] 可从相册选择照片
- [ ] 选择后显示预览
- [ ] 照片自动压缩
- [ ] 可获得 Base64 编码字符串（console 验证）

### 依赖
- 阶段 2 完成
