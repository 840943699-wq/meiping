# 美评 — UI 设计规范 v1.0

---

## 一、设计理念

### 核心关键词
**青春 · 活力 · 简洁 · 正能量**

- **青春**：圆润的边角、明亮的色彩、活泼但不幼稚
- **活力**：克莱因蓝主色调传递自信与力量
- **简洁**：卡片式布局，信息层次清晰，一眼看懂
- **正能量**：避免容貌焦虑，传递「每个人都可以更美」的理念

### 设计参考
- Apple Human Interface Guidelines（iOS 风格）
- 卡片式 Material Design 改良版
- 年轻化美妆/穿搭类 App 设计趋势

## 二、色彩系统

### 主色调：克莱因蓝

```
克莱因蓝 (International Klein Blue)
#002FA7  RGB(0, 47, 167)
```

克莱因蓝由法国艺术家伊夫·克莱因创造，象征**纯粹、无限、自信**。
在颜值评测场景中，蓝色传递**专业、客观、可信赖**的感觉。

### 完整色板

| 色阶 | 色值 | 用途 |
|------|------|------|
| **Primary-700** | `#001F70` | 按钮按下态、深色强调 |
| **Primary-500** | `#002FA7` | 主色调，按钮、标题、图标 |
| **Primary-300** | `#0052CC` | 渐变过渡色、链接 |
| **Primary-100** | `#E8F0FE` | 浅蓝背景、选中态 |
| | | |
| **Neutral-100** | `#F5F7FA` | 页面背景 |
| **Neutral-200** | `#FFFFFF` | 卡片背景 |
| **Neutral-700** | `#1A1A2E` | 主文字色 |
| **Neutral-500** | `#666666` | 次级文字 |
| **Neutral-300** | `#E8E8E8` | 分割线/边框 |
| | | |
| **Success** | `#34C759` | 优势标签、高分 |
| **Warning** | `#FF9500` | 中等分数提示 |
| **Error** | `#FF3B30` | 低分警示（谨慎使用） |

### 配色原则
1. 大面积用浅色背景（白/浅灰），小面积用克莱因蓝点缀
2. 主按钮用克莱因蓝，次按钮用描边样式
3. 梯度评分：高分用蓝色系，中分用橙色系，低分用灰色系
4. 避免大红色 —— 不制造焦虑

## 三、字体规范

### 字体族

```css
font-family: -apple-system, BlinkMacSystemFont,
  "PingFang SC", "Helvetica Neue",
  "Microsoft YaHei", sans-serif;
```

优先使用系统原生中文字体（PingFang SC），确保最佳渲染效果。

### 字号等级

| 等级 | 大小 | 用途 |
|------|------|------|
| **Score** | `72px / Bold` | 综合评分大数字 |
| **XXL** | `48px / Bold` | 页面大标题 |
| **XL** | `28px / Bold` | 卡片标题 |
| **LG** | `20px / Medium` | Tab 标签、按钮文字 |
| **MD** | `16px / Regular` | 正文、评语 |
| **SM** | `14px / Regular` | 辅助说明、时间 |
| **XS** | `12px / Regular` | 标签、角标 |

### 字重

- **Bold (700)**：评分数字、页面标题、按钮
- **Medium (500)**：Tab 标签、卡片标题
- **Regular (400)**：正文、描述

## 四、布局规范

### 基准屏幕
- 设计基准宽度：**375pt**（iPhone SE / 6/7/8 / 12/13 mini）
- 内容安全区：左右各 **20pt** 边距
- 最大内容宽度：**600pt**（桌面端居中显示）

### 间距系统（8pt 基准）

| 等级 | 大小 | 用途 |
|------|------|------|
| **XS** | `4px` | 紧密元素间距 |
| **SM** | `8px` | 标签间距、图标与文字 |
| **MD** | `16px` | 卡片内边距、列表间距 |
| **LG** | `24px` | 段落间距、模块间距 |
| **XL** | `32px` | 页面章节间距 |
| **XXL** | `48px` | 页面上下留白 |

### 圆角规范

| 元素 | 圆角 | 说明 |
|------|------|------|
| 大卡片 | `24px` | 评分卡片、结果卡片 |
| 小卡片 | `16px` | 列表卡片、计划卡片 |
| 按钮 | `12px` | 主按钮、次按钮 |
| 标签/徽章 | `8px` | 风格标签、评分标签 |
| 图片预览 | `16px` | 照片预览区 |

### 触摸规范（iOS HIG）

- 最小触摸目标：**44×44pt**
- 按钮最小高度：**48pt**
- 按钮最小宽度：**120pt**（主按钮）
- 列表项最小高度：**52pt**

## 六、动效规范

### 6.1 页面切换
- 过渡类型：淡入淡出 + 轻微上移
- 持续时间：300ms
- 缓动函数：`ease-out`

### 6.2 评分数字
- 加载完成后数字从 0 滚动到目标分数
- 持续时间：800ms
- 缓动函数：`cubic-bezier(0.25, 0.1, 0.25, 1)`

### 6.3 进度条
- 从 0 宽度过渡到目标宽度
- 持续时间：600ms
- 各维度依次延迟 150ms 展开

### 6.4 按钮反馈
- 按下：scale(0.96) + 颜色加深
- 持续时间：150ms
- 缓动函数：`ease-in-out`

### 6.5 加载动画
- 克莱因蓝圆环旋转 + 文字提示
- 支持不确定进度（旋转动画）和确定进度（百分比）

## 八、响应式策略

### 移动端（< 768px）—— 优先设计
- 单列布局
- 全宽卡片
- 底部固定按钮
- Tab 可横向滚动

### 桌面端（≥ 768px）
- 内容最大宽度 600px 居中
- 双列布局（结果页）
- 卡片放大但保持比例

## 九、视觉检查清单

- [ ] 克莱因蓝使用不过量（面积 < 30%）
- [ ] 所有按钮触摸区域 ≥ 44pt
- [ ] 卡片圆角统一
- [ ] 文字对比度符合可读性（WCAG AA）
- [ ] 评分数字醒目可读
- [ ] 动画流畅不卡顿（60fps）
- [ ] 深色模式下可读（基础兼容即可）
