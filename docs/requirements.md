# 美评 — 产品需求文档 v1.0

---

## 一、产品概述

### 1.1 产品简介
「美评」是一款基于 AI 大模型的颜值分析与形象顾问工具。用户通过拍照或上传照片，获得 AI 给出的专业颜值评分、优劣势分析、变美计划和穿搭建议。

### 1.2 目标用户
- 关注个人形象的普通用户
- 希望通过 AI 获取客观颜值反馈的人群
- 寻求变美灵感和穿搭建议的年轻群体

### 1.3 核心价值
- **客观评分**：AI 多维度分析，给出公正的颜值评价
- **专业建议**：不只打分，更给出可操作的变美指导
- **正能量导向**：帮助用户发现自己的美，拒绝容貌焦虑
- **零门槛使用**：手机浏览器打开即用，无需下载安装

## 二、功能模块

### 2.1 照片采集
- 支持调用手机相机实时拍照
- 支持从手机相册选择已有照片
- 照片预览与重新选择
- 不限制照片类型（单人/多人/侧脸等均可尝试）

### 2.2 AI 颜值分析
- 调用智谱 GLM-4.6V-Flash 视觉模型进行分析
- 分析维度：
  - **五官**（眼睛、鼻子、嘴巴、耳朵）
  - **脸型**（轮廓、比例、对称性）
  - **皮肤**（肤质、肤色、光泽度）
  - **气质**（整体感觉、表现力、感染力）
- 返回结构化 JSON 结果

### 2.3 综合打分（模块1）
- **十分制评分**：0.0 ~ 10.0 分，精确到小数点后一位
- 大号数字展示综合评分
- 各维度评分以进度条/条形图展示
- AI 综合评语

### 2.4 优势分析（模块2）
- 列出 3~5 个面相/五官优势
- 每个优势附简短说明
- 正能量、鼓励性语言

### 2.5 劣势与改进（模块3）
- 列出 2~4 个可优化之处
- 每个劣势附带具体可操作的改进建议
- 建设性语气，避免负面打击

### 2.6 变美计划（模块4）
- **短期计划**（1 周内可执行）
- **中期计划**（1 个月内可执行）
- **长期计划**（3 个月内可执行）
- 涵盖：护肤、妆容、发型、体态、表情管理等

### 2.7 穿搭建议（模块5）
- 根据面部风格判断（甜美型/冷艳型/清新型/大气型等）
- 推荐 2~3 种穿搭风格
- 适合的配色方案
- 具体单品建议

## 三、评分体系

### 3.1 评分标准
| 分数区间 | 等级 | 说明 |
|----------|------|------|
| 9.0 - 10.0 | 惊艳 | 五官精致，气质出众 |
| 8.0 - 8.9 | 优秀 | 颜值在线，有辨识度 |
| 7.0 - 7.9 | 良好 | 整体不错，有小幅提升空间 |
| 6.0 - 6.9 | 中等 | 有基础，可通过打扮大幅提升 |
| 5.0 - 5.9 | 一般 | 建议关注改进建议 |
| 5.0 以下 | 潜力股 | 变美空间很大！ |

### 3.2 评分原则
- AI 应客观公正，不过分抬高也不刻意压低
- 评语正能量、建设性
- 如果照片无法清晰分析人脸，应如实告知

## 四、UI 需求

### 4.1 整体风格
- **年轻化**：圆角、大卡片、活泼但不幼稚
- **简洁**：信息层次清晰，一眼看懂
- **正能量**：配色明亮，避免暗沉

### 4.2 主色调
- 主色：克莱因蓝 `#002FA7`
- 辅助色系：浅蓝渐变、白色背景、浅灰卡片

### 4.3 页面结构
1. **首页**：Logo + Slogan「发现你的美」+ 开始按钮
2. **拍照页**：预览区 + 拍照/相册按钮
3. **加载页**：动画 + 进度提示
4. **结果页**：5 个 Tab 切换（综合/优势/改进/计划/穿搭）

### 4.4 移动端适配
- iPhone 屏幕优先（375pt 宽）
- 大按钮（≥ 44pt 触摸区域）
- 大字体，方便阅读

## 五、非功能需求

### 5.1 性能
- 首屏加载 < 2 秒
- AI 分析加载有明确进度反馈
- 图片压缩后再上传（减少流量消耗）

### 5.2 兼容性
- iOS Safari（iOS 15+）
- Android Chrome
- 桌面端 Chrome / Safari（响应式）

### 5.3 PWA 特性
- 可添加到手机主屏幕
- 全屏模式运行（无浏览器边框）
- 离线缓存基础页面

### 5.4 隐私
- 照片仅用于本次 AI 分析，不存储到任何服务器
- API Key 不对外暴露（仅前端调用，用户自己保管）
